# ⌚ 애플워치 연동 설정 (아이폰 단축어, 1회 설정)

애플워치 운동 기록을 식단운동 앱에 자동 기입합니다.
원리: 워치 데이터는 아이폰 **건강앱**에 쌓임 → **단축어**가 읽어서 앱 URL로 전달 → 앱이 기록.

## 1회 설정 (아이폰에서, 약 3분)

1. **단축어 앱** 열기 → 우상단 `+` (새 단축어)
2. 동작 추가 → 검색 **"운동 찾기"** (또는 "건강 샘플 찾기" → 유형 **운동**)
   - 필터: **시작 날짜** = "오늘" (또는 지난 1일)
3. 동작 추가 → **"각 항목 반복"** (2번 결과에 대해)
   - 반복 안에: **"텍스트"** 동작 →
     `{"date":"운동시작날짜","type":"운동유형","minutes":운동시간(분),"kcal":활성에너지,"distance":거리(m)}`
     (파란 변수로 각각 삽입 — 운동의 세부사항에서 선택)
4. 반복 밖: **"텍스트 결합"** (줄바꿈) → **"텍스트"** 동작으로 감싸기:
   `{"workouts":[결합결과를 콤마로]}`
   ▸ 간단 버전: 운동 1개만 보내려면 3번 텍스트를 그대로 아래 5번에 사용
5. 동작 추가 → **"URL 인코딩"** (4번 텍스트)
6. 동작 추가 → **"URL 열기"**:
   `https://yoojongryol-gif.github.io/diet-fitness/#watch=인코딩된텍스트`
7. 단축어 이름: **"워치→식단앱"** 저장

## 사용법
- 워치로 운동 → 아이폰에서 단축어 **"워치→식단앱"** 탭 → 사파리가 앱 열리며 "⌚ 워치 운동 N건 가져옴" 토스트
- **자동화**(선택): 단축어 앱 → 자동화 → "운동을 종료할 때" → 위 단축어 실행 → 완전 자동

## 수동 백업 경로
단축어가 번거로우면: 앱 운동 탭의 **⌚ 버튼** → JSON 붙여넣기
```json
{"workouts":[{"date":"2026-07-03","type":"걷기","minutes":32,"kcal":180,"distance":2400}],"weight":{"date":"2026-07-03","kg":78.5}}
```

## 참고
- 같은 기록은 여러 번 가져와도 중복 저장 안 됨 (자동 스킵)
- `weight`(체중)도 함께 보낼 수 있음 — 워치/체중계가 건강앱에 쓴 값
- PWA는 애플 정책상 건강앱(HealthKit) 직접 접근 불가 → 단축어가 유일한 무료 브리지
